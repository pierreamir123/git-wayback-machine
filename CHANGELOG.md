# Changelog

All notable changes to the "git-wayback-machine" extension will be documented in this file.

## [0.0.1] - 2026-05-09

### Added
- Initial release of Git Wayback Machine.
- Interactive vertical timeline for file history.
- "Play across commits" replay feature with cinematic animations.
- Multi-author support with distinct color coding for each contributor.
- Pulsing author-specific cursors during history replay.
- Insights Engine providing File Stability Scores and Story Mode narrative.
- Git blame integration with heatmap age indicators.
- Author filtering in the timeline view.
- Support for replaying history from the very first commit ("Restart from Oldest").
- VSCode Activity Bar integration.
- Packaged as an installable `.vsix` for easy distribution.
